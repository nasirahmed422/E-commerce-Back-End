# E-commerce-Back-End

## Purpose
The purpose of this project is to build the back end for an e-commerce site. I took a working Express.js API and configured it to use Sequelize to interact with a MySQL database.

## Built With
* Javascript
* Node
* Express
* MySQL
* Sequelize
